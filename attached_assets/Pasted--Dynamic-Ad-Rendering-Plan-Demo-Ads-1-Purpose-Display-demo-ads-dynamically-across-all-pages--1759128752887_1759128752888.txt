4Ô∏è‚É£ Frontend Rendering Logic

Steps for each ad slot:

Fetch all enabled ads for the given slot from local database.

For each ad:

If adScript exists ‚Üí inject dynamically using dangerouslySetInnerHTML or equivalent.

If bannerImage exists ‚Üí render:

<a href="bannerLink" target="_blank" rel="noopener">
  <img src="bannerImage" width="ad.width || defaultWidth" height="ad.height || defaultHeight" style="object-fit: cover; max-width: 100%;" />
</a>


If width or height = 0 ‚Üí fallback to DEFAULT_AD_SIZES based on slot type.

Make sure the container respects dimensions (network scripts or banners).
